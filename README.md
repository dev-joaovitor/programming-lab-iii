# Programming Lab III

## Summary
The purpose of this project is to exercise fullstack development using Node.js based tools.

## Techs
- Node.js
- Express
- PostgreSQL

## How to run?
1. Install the Node version specified in the `.nvmrc` file
  - If you have the `Node version manager` installed, run `nvm install`
  - else install from the Node.js website or your Linux package manager
2. Run the server
  - Use the script `npm run dev`
